# Redis 基础

# 什么是 Redis？

Redis 是一个基于 C 语言开发的基于内存的键值型 NoSQL 数据库，读写速度快，被广泛用于缓存方向。

有以下几个特征：

  1. 为了满足不同的业务场景，Redis 内置了多种数据类型实现（比如 String、Hash、Sorted Set、Bitmap、HyperLogLog、GEO）。
  2. 单线程，每个命令具备原子性。
  3. 低延迟，速度快（基于内存、IO 多路复用、良好的编码）。
  4. 支持事务、数据持久化、Lua 脚本、多种开箱即用的集群方案（Redis Sentinel、Redis Cluster）。
  5. 没有外部依赖，官方推荐生产环境使用 Linux 部署 Redis。
  6. 支持多语言客户端。

# Redis 为什么这么快？

因为 Redis 内部做了非常多的性能优化，比如：

  1. Redis 基于内存，内存的访问速度是硬盘的上千倍；
  2. Redis 基于 Reactor 模式设计开发了一套高效的事件处理模型，主要是单线程事件循环和 IO 多路复用；

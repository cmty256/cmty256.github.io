ж·ұе…ҘдәҶи§Ј Bearer жЁЎејҸ

# ж·ұе…ҘдәҶи§Ј Bearer жЁЎејҸ

еңЁејҖеҸ‘ Web еә”з”Ёе’Ң RESTful API ж—¶пјҢBearer жЁЎејҸдҪңдёәдёҖз§Қеёёи§Ғзҡ„и®ӨиҜҒж–№ејҸе№ҝжіӣдҪҝз”ЁпјҢе°Өе…¶жҳҜеңЁеҹәдәҺ OAuth 2.0 е’Ң JWTпјҲJSON Web Tokenпјүзҡ„и®ӨиҜҒж–№жЎҲдёӯгҖӮжң¬зҜҮж–Үз« е°Ҷзі»з»ҹжҖ§ең°д»Ӣз»Қ Bearer жЁЎејҸзҡ„зӣёе…ізҹҘиҜҶзӮ№пјҢеё®еҠ©дҪ е…ЁйқўжҺҢжҸЎе…¶ж ёеҝғжҰӮеҝөгҖҒе®һзҺ°ж–№ејҸд»ҘеҸҠеә”з”ЁеңәжҷҜгҖӮ

# д»Җд№ҲжҳҜ Bearer жЁЎејҸпјҹ

Bearer жЁЎејҸжҳҜ OAuth 2.0 ж ҮеҮҶе®ҡд№үзҡ„дёҖз§Қд»ӨзүҢи®ӨиҜҒж–№ејҸгҖӮBearer Token зҡ„еҗ«д№үжҳҜвҖңжҢҒжңүиҖ…д»ӨзүҢвҖқпјҢеҚіпјҡ

  • и°ҒжӢҘжңүиҝҷдёӘд»ӨзүҢпјҢе°ұеҸҜд»ҘеҮӯеҖҹе®ғи®ҝй—®еҸ—дҝқжҠӨзҡ„иө„жәҗгҖӮ
  • е®ғйҖҡеёёз”ЁдәҺеңЁж— зҠ¶жҖҒ HTTP иҜ·жұӮдёӯдј йҖ’з”ЁжҲ·иә«д»ҪжҲ–жҺҲжқғдҝЎжҒҜгҖӮ

Bearer Token зҡ„и®ӨиҜҒдҝЎжҒҜдёҖиҲ¬йҖҡиҝҮ HTTP иҜ·жұӮеӨҙзҡ„ Authorization еӯ—ж®өдј йҖ’пјҢж јејҸеҰӮдёӢпјҡ

Authorization: Bearer <token>
1

е…¶дёӯпјҡ

  • Bearer жҳҜеӣәе®ҡзҡ„еүҚзјҖпјҢз”ЁдәҺж ҮиҜҶи®ӨиҜҒзұ»еһӢгҖӮ
  • <token> жҳҜе…·дҪ“зҡ„д»ӨзүҢпјҲдҫӢеҰӮ JWTпјүгҖӮ

# Bearer Token зҡ„е·ҘдҪңеҺҹзҗҶ

Bearer Token зҡ„е·ҘдҪңжөҒзЁӢеҰӮдёӢпјҡ

  1. з”ЁжҲ·и®ӨиҜҒпјҡз”ЁжҲ·йҖҡиҝҮз”ЁжҲ·еҗҚе’ҢеҜҶз Ғзҷ»еҪ•пјҢжҲ–иҖ…йҖҡиҝҮ OAuth 2.0 зҡ„жҺҲжқғжөҒзЁӢиҺ·еҸ–и®ҝй—®д»ӨзүҢпјҲAccess TokenпјүгҖӮ
  2. еҲҶеҸ‘д»ӨзүҢпјҡжңҚеҠЎеҷЁз”ҹжҲҗд»ӨзүҢе№¶иҝ”еӣһз»ҷе®ўжҲ·з«ҜгҖӮ
  3. жҗәеёҰд»ӨзүҢи®ҝй—®иө„жәҗпјҡе®ўжҲ·з«ҜеңЁеҗҺз»ӯзҡ„иҜ·жұӮдёӯпјҢйҖҡиҝҮ Authorization еӨҙжҗәеёҰд»ӨзүҢгҖӮ
  4. йӘҢиҜҒд»ӨзүҢпјҡжңҚеҠЎеҷЁйӘҢиҜҒд»ӨзүҢзҡ„жңүж•ҲжҖ§пјҲеҰӮзӯҫеҗҚж ЎйӘҢгҖҒиҝҮжңҹж—¶й—ҙзӯүпјүгҖӮ
  5. иҝ”еӣһз»“жһңпјҡйӘҢиҜҒжҲҗеҠҹеҗҺпјҢжңҚеҠЎеҷЁе…Ғи®ёи®ҝй—®еҸ—дҝқжҠӨиө„жәҗгҖӮ

# дёәд»Җд№ҲиҰҒдҪҝз”Ё Bearer еүҚзјҖпјҹ

Bearer еүҚзјҖжҳҜдёҖз§Қж ҮеҮҶеҢ–зҡ„зәҰе®ҡпјҢдёәи®ӨиҜҒзі»з»ҹеёҰжқҘд»ҘдёӢдјҳеҠҝпјҡ

  1. жҳҺзЎ®и®ӨиҜҒзұ»еһӢпјҡ
    • Bearer зӣҙи§ӮиЎЁжҳҺиҝҷжҳҜеҹәдәҺ OAuth 2.0 зҡ„ Token и®ӨиҜҒпјҢиҖҢдёҚжҳҜе…¶д»–зұ»еһӢзҡ„и®ӨиҜҒпјҲеҰӮ Basic и®ӨиҜҒпјүгҖӮ
  2. еўһејәе…је®№жҖ§пјҡ
    • и®ёеӨҡж ҮеҮҶеҢ–и®ӨиҜҒеә“пјҲеҰӮ Spring SecurityгҖҒPassport.js зӯүпјүй»ҳи®Өж”ҜжҢҒ Bearer TokenгҖӮеҰӮжһңзңҒз•Ҙ BearerпјҢеҸҜиғҪеҜјиҮҙе…је®№жҖ§й—®йўҳгҖӮ
  3. жҸҗй«ҳеҸҜиҜ»жҖ§пјҡ
    • Bearer жҸҗдҫӣдәҶдёҖз§Қз»ҹдёҖзҡ„ж јејҸпјҢдҪҝејҖеҸ‘иҖ…е’Ңе·Ҙе…·иғҪеӨҹеҝ«йҖҹиҜҶеҲ«д»ӨзүҢзұ»еһӢгҖӮ

# дҪҝз”Ё Bearer Token зҡ„еңәжҷҜ

Bearer Token дё»иҰҒз”ЁдәҺд»ҘдёӢеңәжҷҜпјҡ

  1. API и®ӨиҜҒдёҺжҺҲжқғ
    • е®ўжҲ·з«ҜйҖҡиҝҮд»ӨзүҢи®ҝй—®еҸ—дҝқжҠӨзҡ„ APIпјҲеҰӮз”ЁжҲ·ж•°жҚ®гҖҒи®ўеҚ•дҝЎжҒҜзӯүпјүгҖӮ
  2. еҲҶеёғејҸзі»з»ҹи®ӨиҜҒ
    • еҫ®жңҚеҠЎд№Ӣй—ҙйҖҡиҝҮ Bearer Token дј йҖ’з”ЁжҲ·иә«д»Ҫе’ҢжқғйҷҗгҖӮ
  3. з¬¬дёүж–№еә”з”ЁжҺҲжқғ
    • еңЁ OAuth 2.0 дёӯпјҢз¬¬дёүж–№еә”з”ЁдҪҝз”Ё Bearer Token и®ҝй—®з”ЁжҲ·жҺҲжқғзҡ„иө„жәҗгҖӮ

# Bearer Token зҡ„е®үе…ЁжҖ§

Bearer Token жҳҜдёҖз§Қж•Ҹж„ҹдҝЎжҒҜпјҢиӢҘиў«жі„йңІпјҢж”»еҮ»иҖ…еҚіеҸҜеҶ’е……еҗҲжі•з”ЁжҲ·и®ҝй—®еҸ—дҝқжҠӨиө„жәҗгҖӮеӣ жӯӨпјҢд»ҘдёӢжҳҜдёҖдәӣе®үе…Ёе»әи®®пјҡ

  1. дҪҝз”Ё HTTPS
    • зЎ®дҝқжүҖжңүдј иҫ“д»ӨзүҢзҡ„иҜ·жұӮйғҪдҪҝз”Ё HTTPSпјҢйҳІжӯўд»ӨзүҢеңЁдј иҫ“иҝҮзЁӢдёӯиў«зӘғеҗ¬гҖӮ
  2. и®ҫзҪ®иҝҮжңҹж—¶й—ҙ
    • Bearer Token еә”и®ҫзҪ®еҗҲзҗҶзҡ„иҝҮжңҹж—¶й—ҙпјҢйҷҚдҪҺиў«зӣ—з”Ёзҡ„йЈҺйҷ©гҖӮ
  3. дҪҝз”ЁеҲ·ж–°д»ӨзүҢпјҲRefresh Tokenпјү
    • еҜ№дәҺй•ҝжңҹдјҡиҜқпјҢеҸҜд»ҘйҖҡиҝҮзҹӯжңҹжңүж•Ҳзҡ„ Bearer Token й…ҚеҗҲеҲ·ж–°д»ӨзүҢзҡ„жңәеҲ¶жқҘжҸҗй«ҳе®үе…ЁжҖ§гҖӮ
  4. IP е’Ңи®ҫеӨҮз»‘е®ҡ
    • еҜ№д»ӨзүҢзҡ„дҪҝз”ЁеңәжҷҜиҝӣиЎҢйҷҗеҲ¶пјҢеҰӮз»‘е®ҡз”ЁжҲ· IP жҲ–и®ҫеӨҮгҖӮ
  5. еӯҳеӮЁж•Ҹж„ҹдҝЎжҒҜж—¶еҠ еҜҶ
    • еңЁе®ўжҲ·з«ҜжҲ–жңҚеҠЎеҷЁдёӯеӯҳеӮЁд»ӨзүҢж—¶пјҢйҮҮз”ЁеҠ еҜҶжңәеҲ¶дҝқжҠӨж•°жҚ®гҖӮ

# Bearer Token зҡ„еұҖйҷҗжҖ§

иҷҪз„¶ Bearer Token жҳҜдёҖз§Қй«ҳж•Ҳзҡ„и®ӨиҜҒж–№ејҸпјҢдҪҶд№ҹеӯҳеңЁдёҖдәӣеұҖйҷҗжҖ§пјҡ

  1. ж— з»‘е®ҡзү№жҖ§пјҡ
    • Bearer Token жІЎжңүз»‘е®ҡеҲ°зү№е®ҡзҡ„з”ЁжҲ·жҲ–и®ҫеӨҮпјҢдёҖж—Ұжі„йңІпјҢд»»дҪ•дәәйғҪеҸҜд»ҘдҪҝз”ЁгҖӮ

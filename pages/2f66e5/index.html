# RocketMQ е…Ҙй—Ё

и§Ҷйў‘ең°еқҖпјҡ

еҚғй”Ӣж•ҷиӮІRocketMQе…ЁеҘ—и§Ҷйў‘ж•ҷзЁӢпјҢеҝ«йҖҹжҺҢжҸЎMQж¶ҲжҒҜдёӯй—ҙд»¶ (opens new window)

# д»Җд№ҲжҳҜ MQ пјҹ

Message QueueпјҲж¶ҲжҒҜ йҳҹеҲ—пјүпјҢд»Һеӯ—йқўдёҠзҗҶи§ЈпјҡйҰ–е…Ҳе®ғжҳҜдёҖдёӘйҳҹеҲ—гҖӮFIFO е…Ҳиҝӣе…ҲеҮәзҡ„ж•°жҚ®з»“жһ„ вҖ”вҖ” йҳҹеҲ—гҖӮ

ж¶ҲжҒҜйҳҹеҲ—е°ұжҳҜжүҖи°“зҡ„еӯҳж”ҫж¶ҲжҒҜзҡ„йҳҹеҲ—гҖӮ

ж¶ҲжҒҜйҳҹеҲ—и§ЈеҶізҡ„дёҚжҳҜеӯҳж”ҫж¶ҲжҒҜзҡ„йҳҹеҲ—зҡ„зӣ®зҡ„пјҢи§ЈеҶізҡ„жҳҜйҖҡдҝЎй—®йўҳгҖӮ

1гҖҒеҗҢжӯҘйҖҡдҝЎжғ…еҶөдёӢ

жҜ”еҰӮд»Ҙз”өе•Ҷи®ўеҚ•зі»з»ҹдёәдҫӢпјҢеҰӮжһңеҗ„жңҚеҠЎд№Ӣй—ҙдҪҝз”ЁеҗҢжӯҘйҖҡдҝЎпјҢдёҚд»…иҖ—ж—¶иҫғд№…пјҢдё”иҝҮзЁӢдёӯеҸ—еҲ°зҪ‘з»ңжіўеҠЁзҡ„еҪұе“ҚпјҢдёҚиғҪдҝқиҜҒй«ҳжҲҗеҠҹзҺҮгҖӮеӣ жӯӨпјҢдҪҝз”ЁејӮжӯҘзҡ„йҖҡдҝЎж–№ејҸеҜ№жһ¶жһ„иҝӣиЎҢж”№йҖ гҖӮ

2гҖҒдҪҝз”Ёж¶ҲжҒҜйҳҹеҲ—еҗҺгҖӮејӮжӯҘйҖҡдҝЎзҡ„жғ…еҶөдёӢ

  • дҪҝвҪӨејӮжӯҘзҡ„йҖҡдҝЎж–№ејҸеҜ№жЁЎеқ—й—ҙзҡ„и°ғвҪӨиҝӣиЎҢи§ЈиҖҰпјҢеҸҜд»Ҙеҝ«йҖҹзҡ„жҸҗеҚҮзі»з»ҹзҡ„еҗһеҗҗйҮҸгҖӮ
  • дёҠжёёжү§иЎҢе®Ңж¶ҲжҒҜзҡ„еҸ‘йҖҒдёҡеҠЎеҗҺз«ӢеҚіиҺ·еҫ—з»“жһңпјҢдёӢжёёеӨҡдёӘжңҚеҠЎи®ўйҳ…еҲ°ж¶ҲжҒҜеҗҺеҗ„вҫғж¶Ҳиҙ№гҖӮ
  • йҖҡиҝҮж¶ҲжҒҜйҳҹеҲ—пјҢеұҸи”Ҫеә•еұӮзҡ„йҖҡдҝЎеҚҸи®®пјҢдҪҝеҫ—и§Ји—•е’Ңе№¶иЎҢж¶Ҳиҙ№еҫ—д»Ҙе®һзҺ°гҖӮ

# RocketMQ зҡ„еҹәжң¬жҰӮеҝө

# жҠҖжңҜжһ¶жһ„

RocketMQ жһ¶жһ„дёҠдё»иҰҒеҲҶдёәеӣӣйғЁеҲҶпјҢеҰӮдёҠеӣҫжүҖзӨәпјҡ

  1. Producerпјҡж¶ҲжҒҜеҸ‘еёғзҡ„в»ҶвҫҠпјҢвҪҖжҢҒеҲҶеёғејҸйӣҶзҫӨвҪ…ејҸйғЁзҪІгҖӮProducer йҖҡиҝҮ MQ зҡ„иҙҹиҪҪеқҮиЎЎжЁЎеқ—йҖүжӢ©зӣёеә”зҡ„ Broker йӣҶзҫӨйҳҹеҲ—иҝӣиЎҢж¶ҲжҒҜжҠ•йҖ’пјҢжҠ•йҖ’зҡ„иҝҮзЁӢвҪҖжҢҒеҝ«йҖҹеӨұиҙҘе№¶дё”дҪҺе»¶иҝҹгҖӮ

  2. Consumerпјҡж¶ҲжҒҜж¶Ҳиҙ№зҡ„и§’иүІпјҢж”ҜжҢҒеҲҶеёғејҸйӣҶзҫӨж–№ејҸйғЁзҪІгҖӮвҪҖжҢҒд»Ҙ push жҺЁпјҢpull жӢүдёӨз§ҚжЁЎејҸеҜ№ж¶ҲжҒҜиҝӣиЎҢж¶Ҳиҙ№гҖӮеҗҢж—¶д№ҹвҪҖжҢҒйӣҶзҫӨж–№ејҸе’Ңе№ҝж’ӯж–№ејҸзҡ„ж¶Ҳиҙ№пјҢе®ғжҸҗдҫӣе®һж—¶ж¶ҲжҒҜи®ўйҳ…жңәеҲ¶пјҢеҸҜд»Ҙж»Ўи¶іеӨ§еӨҡж•°з”ЁжҲ·зҡ„йңҖжұӮгҖӮ

  3. NameServerпјҡNameServer жҳҜвјҖдёӘйқһеёёз®ҖеҚ•зҡ„ Topic и·Ҝз”ұжіЁеҶҢдёӯеҝғпјҢе…¶в»ҶвҫҠзұ»дјј Dubbo дёӯзҡ„ zookeeperпјҢж”ҜжҢҒ Broker зҡ„еҠЁжҖҒжіЁеҶҢдёҺеҸ‘зҺ°гҖӮ

    • дё»иҰҒеҢ…жӢ¬дёӨдёӘеҠҹиғҪпјҡBroker з®ЎзҗҶпјҢNameServer жҺҘж”¶ Broker йӣҶзҫӨзҡ„жіЁеҶҢдҝЎжҒҜе№¶дё”дҝқеӯҳдёӢжқҘдҪңдёәи·Ҝз”ұдҝЎжҒҜзҡ„еҹәжң¬ж•°жҚ®гҖӮз„¶еҗҺжҸҗдҫӣеҝғи·іжЈҖжөӢжңәеҲ¶пјҢжЈҖжҹҘ Broker жҳҜеҗҰиҝҳеӯҳжҙ»пјӣи·Ҝз”ұдҝЎжҒҜз®ЎзҗҶпјҢжҜҸдёӘ NameServer е°Ҷдҝқеӯҳе…ідәҺ Broker йӣҶзҫӨзҡ„ж•ҙдёӘи·Ҝз”ұдҝЎжҒҜе’ҢвҪӨдәҺе®ўжҲ·з«ҜжҹҘиҜўзҡ„йҳҹеҲ—дҝЎжҒҜгҖӮ

    • з„¶еҗҺ Producer е’Ң Conumser йҖҡиҝҮ NameServer е°ұеҸҜд»ҘзҹҘйҒ“ж•ҙдёӘ Broker йӣҶзҫӨзҡ„и·Ҝз”ұдҝЎжҒҜпјҢд»ҺвҪҪиҝӣвҫҸж¶ҲжҒҜзҡ„жҠ•йҖ’е’Ңж¶Ҳиҙ№гҖӮ

    • NameServer йҖҡеёёд№ҹжҳҜйӣҶзҫӨзҡ„вҪ…ејҸйғЁзҪІпјҢеҗ„е®һдҫӢй—ҙзӣёдә’дёҚиҝӣвҫҸдҝЎжҒҜйҖҡи®ҜгҖӮеӣ дёә Broker жҳҜеҗ‘жҜҸвјҖеҸ° NameServer жіЁеҶҢвҫғвј°зҡ„и·Ҝз”ұдҝЎжҒҜпјҢжүҖд»ҘжҜҸвјҖдёӘ NameServer е®һдҫӢдёҠвҫҜйғҪдҝқеӯҳвјҖд»Ҫе®Ңж•ҙзҡ„и·Ҝз”ұдҝЎжҒҜгҖӮ

    • еҪ“жҹҗдёӘ NameServer еӣ жҹҗз§ҚеҺҹеӣ дёӢзәҝдәҶпјҢBroker д»Қз„¶еҸҜд»Ҙеҗ‘е…¶е®ғ NameServer еҗҢжӯҘе…¶и·Ҝз”ұдҝЎжҒҜпјҢProducer,Consumer д»Қз„¶еҸҜд»ҘеҠЁжҖҒж„ҹзҹҘ Broker зҡ„и·Ҝз”ұзҡ„дҝЎжҒҜгҖӮ

  4. BrokerServerпјҡBroker дё»иҰҒиҙҹиҙЈж¶ҲжҒҜзҡ„еӯҳеӮЁгҖҒжҠ•йҖ’е’ҢжҹҘиҜўд»ҘеҸҠжңҚеҠЎй«ҳеҸҜз”ЁдҝқиҜҒпјҢдёәдәҶе®һзҺ°иҝҷдәӣеҠҹиғҪпјҢBroker еҢ…еҗ«дәҶд»ҘдёӢеҮ дёӘйҮҚиҰҒвјҰжЁЎеқ—гҖӮ

    • Remoting ModuleпјҲиҙҹиҙЈеӨ„зҗҶиҜ·жұӮпјүпјҡж•ҙдёӘ Broker зҡ„е®һдҪ“пјҢиҙҹиҙЈеӨ„зҗҶжқҘиҮӘ clients з«Ҝзҡ„иҜ·жұӮгҖӮ
    • Client ManagerпјҲиҙҹиҙЈз®ЎзҗҶи®ўйҳ…дҝЎжҒҜпјүпјҡиҙҹиҙЈз®ЎзҗҶе®ўжҲ·з«Ҝ(Producer/Consumer)е’Ңз»ҙжҠӨ Consumer зҡ„ Topic и®ўйҳ…дҝЎжҒҜгҖӮ
    • Store ServiceпјҡжҸҗдҫӣвҪ…дҫҝз®ҖеҚ•зҡ„ API жҺҘвјқеӨ„зҗҶж¶ҲжҒҜеӯҳеӮЁеҲ°зү©зҗҶзЎ¬зӣҳе’ҢжҹҘиҜўеҠҹиғҪгҖӮ
    • HA ServiceпјҲжҸҗдҫӣж•°жҚ®еҗҢжӯҘеҠҹиғҪпјүпјҡвҫјеҸҜвҪӨжңҚеҠЎпјҢжҸҗдҫӣ Master Broker е’Ң Slave Broker д№Ӣй—ҙзҡ„ж•°жҚ®еҗҢжӯҘеҠҹиғҪгҖӮ
    • Index ServiceпјҲзҙўеј•жңҚеҠЎпјүпјҡж №жҚ®зү№е®ҡзҡ„ Message key еҜ№жҠ•йҖ’еҲ° Broker зҡ„ж¶ҲжҒҜиҝӣвҫҸзҙўеј•жңҚеҠЎпјҢд»ҘжҸҗдҫӣж¶ҲжҒҜзҡ„еҝ«йҖҹжҹҘиҜўгҖӮ

з®ҖиҖҢиЁҖд№Ӣ

NameServer жңҚеҠЎеҷЁеңЁж¶ҲжҒҜеҸ‘йҖҒж—¶е……еҪ“дәҶи·Ҝз”ұдҝЎжҒҜзҡ„жҸҗдҫӣиҖ…пјҢеё®еҠ©з”ҹдә§иҖ…жүҫеҲ°жӯЈзЎ®зҡ„ Broker жқҘеҸ‘йҖҒж¶ҲжҒҜгҖӮе®ғиө·еҲ°дәҶжіЁеҶҢе’ҢеҸ‘зҺ° Broker зҡ„дҪңз”ЁпјҢд»ҘзЎ®дҝқж¶ҲжҒҜиғҪеӨҹеҮҶзЎ®ең°и·Ҝз”ұеҲ°зӣ®ж ҮдҪҚзҪ®гҖӮ

еҪ“гҖҗз”ҹдә§иҖ…гҖ‘еҸ‘йҖҒж¶ҲжҒҜж—¶пјҢе®ғдјҡж №жҚ® Topic йҖүжӢ©еҗҲйҖӮзҡ„ Broker жқҘеҸ‘йҖҒж¶ҲжҒҜгҖӮBroker иҙҹиҙЈеӯҳеӮЁе’Ңз®ЎзҗҶж¶ҲжҒҜпјҢжҜҸдёӘ Broker дёҠеҸҜд»Ҙз®ЎзҗҶеӨҡдёӘ TopicгҖӮеӣ жӯӨпјҢTopic жҳҜеңЁ Broker дёҠеҲӣе»әе’Ңз®ЎзҗҶзҡ„пјҢиҖҢдёҚжҳҜеңЁ NameServer жңҚеҠЎеҷЁдёҠгҖӮ

# йғЁзҪІжһ¶жһ„

RocketMQ зҪ‘з»ңйғЁзҪІзү№зӮ№

  1. NameServer жҳҜвјҖдёӘвјҸд№ҺвҪҶзҠ¶жҖҒиҠӮзӮ№пјҢеҸҜйӣҶзҫӨйғЁзҪІпјҢиҠӮзӮ№д№Ӣй—ҙвҪҶд»»дҪ•дҝЎжҒҜеҗҢжӯҘгҖӮ

  2. Broker йғЁзҪІзӣёеҜ№еӨҚжқӮпјҢBroker еҲҶдёә Master дёҺ SlaveпјҢвјҖдёӘ Master еҸҜд»ҘеҜ№еә”еӨҡдёӘ SlaveпјҢдҪҶжҳҜвјҖдёӘ Slave еҸӘиғҪеҜ№еә”вјҖдёӘ MasterгҖӮ

    • Master дёҺ Slave зҡ„еҜ№еә”е…ізі»йҖҡиҝҮжҢҮе®ҡзӣёеҗҢзҡ„ BrokerNameпјҢдёҚеҗҢзҡ„ BrokerId жқҘе®ҡд№үпјҢBrokerId дёә 0 иЎЁзӨә MasterпјҢвҫ® 0 иЎЁзӨә SlaveгҖӮMaster д№ҹеҸҜд»ҘйғЁзҪІеӨҡдёӘгҖӮ

    • жҜҸдёӘ Broker дёҺ NameServer йӣҶзҫӨдёӯзҡ„жүҖжңүиҠӮзӮ№е»әз«Ӣй•ҝиҝһжҺҘпјҢе®ҡж—¶жіЁеҶҢ Topic дҝЎжҒҜеҲ°жүҖжңү NameServerгҖӮ

    • жіЁж„ҸпјҡеҪ“еүҚ RocketMQ зүҲжң¬еңЁйғЁзҪІжһ¶жһ„дёҠвҪҖжҢҒвјҖ Master еӨҡ SlaveпјҢдҪҶеҸӘжңү BrokerId=1 зҡ„д»ҺжңҚеҠЎеҷЁжүҚдјҡеҸӮдёҺж¶ҲжҒҜзҡ„иҜ»иҙҹиҪҪгҖӮ

  3. Producer дёҺ NameServer йӣҶзҫӨдёӯзҡ„е…¶дёӯвјҖдёӘиҠӮзӮ№пјҲйҡҸжңәйҖүжӢ©пјүе»әвҪҙв»“иҝһжҺҘпјҢе®ҡжңҹд»Һ NameServer иҺ·еҸ– Topic и·Ҝз”ұдҝЎжҒҜпјҢе№¶еҗ‘жҸҗдҫӣ Topic жңҚеҠЎзҡ„ Master е»әвҪҙв»“иҝһжҺҘпјҢдё”е®ҡж—¶еҗ‘ Master еҸ‘йҖҒвјји·ігҖӮProducer е®Ңе…ЁвҪҶзҠ¶жҖҒпјҢеҸҜйӣҶзҫӨйғЁзҪІгҖӮ

  4. Consumer дёҺ NameServer йӣҶзҫӨдёӯзҡ„е…¶дёӯвјҖдёӘиҠӮзӮ№пјҲйҡҸжңәйҖүжӢ©пјүе»әвҪҙв»“иҝһжҺҘпјҢе®ҡжңҹд»Һ NameServer иҺ·еҸ– Topic и·Ҝз”ұдҝЎжҒҜпјҢе№¶еҗ‘жҸҗдҫӣ Topic жңҚеҠЎзҡ„ MasterгҖҒSlave е»әвҪҙв»“иҝһжҺҘпјҢдё”е®ҡж—¶еҗ‘ MasterгҖҒSlave еҸ‘йҖҒвјји·ігҖӮConsumer ж—ўеҸҜд»Ҙд»Һ Master и®ўйҳ…ж¶ҲжҒҜпјҢд№ҹеҸҜд»Ҙд»Һ Slave и®ўйҳ…ж¶ҲжҒҜпјҢж¶Ҳиҙ№иҖ…еңЁеҗ‘ Master жӢүеҸ–ж¶ҲжҒҜж—¶пјҢMaster жңҚеҠЎеҷЁдјҡж №жҚ®жӢүеҸ–еҒҸз§»йҮҸдёҺжңҖвјӨеҒҸз§»йҮҸзҡ„и·қзҰ»пјҲеҲӨж–ӯжҳҜеҗҰиҜ»вҪјж¶ҲжҒҜпјҢдә§вҪЈиҜ» I/OпјүпјҢд»ҘеҸҠд»ҺжңҚеҠЎеҷЁжҳҜеҗҰеҸҜиҜ»зӯүеӣ зҙ е»әи®®дёӢвјҖж¬ЎжҳҜд»Һ Master иҝҳжҳҜ Slave жӢүеҸ–гҖӮ

з»“еҗҲйғЁзҪІжһ¶жһ„еӣҫпјҢжҸҸиҝ°йӣҶзҫӨвјҜдҪңжөҒзЁӢ

  1. еҗҜеҠЁ NameServerпјҢNameServer иө·жқҘеҗҺзӣ‘еҗ¬з«ҜвјқпјҢзӯүеҫ… BrokerгҖҒProducerгҖҒConsumer иҝһдёҠжқҘпјҢзӣёеҪ“дәҺвјҖдёӘи·Ҝз”ұжҺ§еҲ¶дёӯеҝғгҖӮ
  2. Broker еҗҜеҠЁпјҢи·ҹжүҖжңүзҡ„ NameServer дҝқжҢҒв»“иҝһжҺҘпјҢе®ҡж—¶еҸ‘йҖҒвјји·іеҢ…гҖӮ
    • еҝғи·іеҢ…дёӯеҢ…еҗ«еҪ“еүҚ Broker дҝЎжҒҜ(IP+з«Ҝвјқзӯү)д»ҘеҸҠеӯҳеӮЁжүҖжңү Topic дҝЎжҒҜгҖӮ
    • жіЁеҶҢжҲҗеҠҹеҗҺпјҢNameServer йӣҶзҫӨдёӯе°ұжңү Topic и·ҹ Broker зҡ„жҳ е°„е…ізі»гҖӮ
    • ж”¶еҸ‘ж¶ҲжҒҜеүҚпјҢе…ҲеҲӣе»ә TopicпјҢеҲӣе»ә Topic ж—¶йңҖиҰҒжҢҮе®ҡиҜҘ Topic иҰҒеӯҳеӮЁеңЁе“Әдәӣ Broker дёҠпјҢд№ҹеҸҜд»ҘеңЁеҸ‘йҖҒж¶ҲжҒҜж—¶вҫғеҠЁеҲӣе»ә TopicгҖӮ
  3. Producer еҸ‘йҖҒж¶ҲжҒҜпјҢеҗҜеҠЁж—¶е…Ҳи·ҹ NameServer йӣҶзҫӨдёӯзҡ„е…¶дёӯвјҖеҸ°е»әвҪҙв»“иҝһжҺҘпјҢе№¶д»Һ NameServer дёӯиҺ·еҸ–еҪ“еүҚеҸ‘йҖҒзҡ„ Topic еӯҳеңЁе“Әдәӣ Broker дёҠпјҢиҪ®иҜўд»ҺйҳҹеҲ—еҲ—иЎЁдёӯйҖүжӢ©вјҖдёӘйҳҹеҲ—пјҢз„¶еҗҺдёҺйҳҹеҲ—жүҖеңЁзҡ„ Broker е»әвҪҙв»“иҝһжҺҘд»ҺвҪҪеҗ‘ Broker еҸ‘ж¶ҲжҒҜгҖӮ
  4. Consumer и·ҹ Producer зұ»дјјпјҢи·ҹе…¶дёӯвјҖеҸ° NameServer е»әвҪҙв»“иҝһжҺҘпјҢиҺ·еҸ–еҪ“еүҚи®ўйҳ…зҡ„ Topic еӯҳеңЁе“Әдәӣ Broker дёҠпјҢз„¶еҗҺзӣҙжҺҘи·ҹ Broker е»әвҪҙиҝһжҺҘйҖҡйҒ“пјҢејҖе§Ӣж¶Ҳиҙ№ж¶ҲжҒҜгҖӮ

RocketMQ е®ҳвҪ…жҸҗдҫӣдәҶдёүз§ҚйӣҶзҫӨжҗӯе»әвҪ…ејҸгҖӮ

# 2 дё» 2 д»ҺејӮжӯҘйҖҡдҝЎж–№ејҸ

дҪҝз”ЁејӮжӯҘж–№ејҸиҝӣиЎҢдё»д»Һд№Ӣй—ҙзҡ„ж•°жҚ®еӨҚеҲ¶пјҢеҗһеҗҗйҮҸеӨ§пјҢдҪҶеҸҜиғҪдјҡдёўж¶ҲжҒҜпјҲеӣ дёәејӮжӯҘзҡ„жғ…еҶөдёӢпјҢд»ҺжңҚеҠЎеҷЁ slave еҸҜиғҪдјҡеӨҚеҲ¶еӨұиҙҘпјүгҖӮ

дҪҝз”Ё conf/2m-2s-async ж–Үд»¶еӨ№еҶ…зҡ„й…ҚзҪ®ж–Үд»¶еҒҡйӣҶзҫӨй…ҚзҪ®гҖӮ

# 2 дё» 2 д»ҺеҗҢжӯҘйҖҡдҝЎж–№ејҸ

# Dledger й«ҳеҸҜз”ЁйӣҶзҫӨ

дёҠиҝ°дёүз§Қе®ҳж–№жҸҗдҫӣзҡ„йӣҶзҫӨжІЎеҠһжі•е®һзҺ°вҫјеҸҜз”ЁпјҢеҚіеңЁ master иҠӮзӮ№жҢӮжҺүеҗҺпјҢslave иҠӮзӮ№жІЎеҠһжі•вҫғеҠЁиў«йҖүдёҫдёәж–°зҡ„ masterпјҢйңҖиҰҒдәәе·Ҙе®һзҺ°гҖӮ

RocketMQ еңЁ 4.5 зүҲжң¬д№ӢеҗҺеј•вјҠдәҶз¬¬дёүж–№зҡ„ Dleger й«ҳеҸҜз”ЁйӣҶзҫӨгҖӮ

        // Specify name server addresses.
        producer.setNamesrvAddr("192.168.194.133:9876");
        // Launch the instance.
        producer.start();

        producer.setRetryTimesWhenSendAsyncFailed(0); // и®ҫзҪ®ејӮжӯҘеҸ‘йҖҒеӨұиҙҘж—¶зҡ„йҮҚиҜ•ж¬Ўж•°

        int messageCount = 100;
        final CountDownLatch countDownLatch = new CountDownLatch(messageCount);

        for (int i = 0; i < messageCount; i++) {
            try {
                final int index = i;
                Message msg = new Message("Jodie_topic_1023",
                        "TagA",
                        "OrderID188",
                        "Hello world".getBytes(RemotingHelper.DEFAULT_CHARSET));
                producer.send(msg, new SendCallback() {
                    // е®ҡд№үеӣһи°ғеҮҪж•°
                    @Override
                    public void onSuccess(SendResult sendResult) {
                        countDownLatch.countDown();
                        System.out.printf("%-10d OK %s %n", index, sendResult.getMsgId());
                    }

                    @Override
                    public void onException(Throwable e) {
                        countDownLatch.countDown();
                        System.out.printf("%-10d Exception %s %n", index, e);
                        e.printStackTrace();
                    }
                });
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
        
        System.out.println("============="); // дјҡеңЁж¶ҲжҒҜеҸҚйҰҲеүҚжү“еҚ°
        countDownLatch.await(5, TimeUnit.SECONDS); // зӯүеҫ…е…ЁйғЁж¶ҲжҒҜиҝ”еӣһе®ҢжҲҗ
        producer.shutdown();
    }

}

public class TransactionListenerImpl implements TransactionListener {
    /**
     * When send transactional prepare(half) message succeed, this method will be invoked to execute local transaction.
     * еҪ“еҸ‘йҖҒдәӢеҠЎеҮҶеӨҮ(еҚҠ)ж¶ҲжҒҜжҲҗеҠҹж—¶пјҢе°Ҷи°ғз”ЁжӯӨж–№123жі• жү§иЎҢ123жң¬ең°дәӢеҠЎгҖӮ

     * @param msg Half(prepare) message
     * @param arg Custom business parameter иҮӘе®ҡд№үдёҡеҠЎеҸӮж•°
     * @return дәӢеҠЎзҠ¶жҖҒ
     */
    @Override
    public LocalTransactionState executeLocalTransaction(Message msg, Object arg) {
        String tags = msg.getTags();
        if(StringUtils.contains(tags,"TagA")){
            return LocalTransactionState.COMMIT_MESSAGE;
        }else if(StringUtils.contains(tags,"TagB")){
            return LocalTransactionState.ROLLBACK_MESSAGE;
        }else{
            return LocalTransactionState.UNKNOW;
        }
    }

    /**
     * еӣһжҹҘжң¬ең°дәӢеҠЎ
     *
     * @param msg Check message
     * @return Transaction state
     */
    @Override
    public LocalTransactionState checkLocalTransaction(MessageExt msg) {
        String tags = msg.getTags();
        if(StringUtils.contains(tags,"TagC")){
            return LocalTransactionState.COMMIT_MESSAGE;
        }else if(StringUtils.contains(tags,"TagD")){
            return LocalTransactionState.ROLLBACK_MESSAGE;
        }else{
            return LocalTransactionState.UNKNOW;
        }
    }
}
